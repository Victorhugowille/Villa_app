# Prompt: Correção de Cálculo no Carrinho

**Data:** 28/10/2025
**Contexto:** Problema crítico onde o valor dos adicionais (ex: "Bacon Extra") não estava somando no total do pedido.

## 🗣️ Prompt do Usuário
"O `CartProvider` não está somando o valor dos adicionais no `totalAmount`.
Quando adiciono um Hambúrguer (R$ 30) com Bacon (R$ 5), o total fica R$ 30 em vez de R$ 35.
Arrume a função `totalPrice`."

## 🤖 Resposta (Lógica de Cálculo)

O erro estava na falta de iteração sobre a lista de `selectedAdicionais`.

### Código Corrigido: `lib/data/app_data.dart` (Trecho)

```dart
class CartItem {
  // ... propriedades

  double get totalPrice {
    // Soma o preço base do produto + a soma de todos os adicionais x quantidade
    double priceAdicionais = selectedAdicionais.fold(
      0.0, 
      (sum, item) => sum + (item.adicional.price * item.quantity)
    );
    
    return (product.price + priceAdicionais) * quantity;
  }
}
