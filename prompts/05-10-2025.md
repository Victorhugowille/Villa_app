# Prompt: Implementação do Tema VillaBistrô

**Data:** 05/10/2025
**Contexto:** Refinamento da UI para abandonar o azul padrão do Flutter e adotar a identidade "Verde e Dourado" do restaurante.

## 🗣️ Prompt do Usuário
"Quero configurar o tema do app com as cores exatas da marca.
Fundo: #1E392A (Verde Escuro)
Texto/Ícones: #F0E6D1 (Creme)
Destaques/Botões: #D4A373 (Dourado/Bronze)

Crie o `ThemeProvider` para gerenciar isso e permitir alternar entre temas se necessário."

## 🤖 Resposta (Theme Configuration)

### Código: `lib/providers/theme_provider.dart`

```dart
class AppThemes {
  static final villaTheme = ThemeData(
    scaffoldBackgroundColor: Color(0xFF1E392A), // Deep Green
    primaryColor: Color(0xFFD4A373), // Gold
    colorScheme: ColorScheme.dark(
      primary: Color(0xFFD4A373),
      secondary: Color(0xFFF0E6D1), // Cream
      surface: Color(0xFF2A4B3A), // Lighter Green for Cards
    ),
    // ... configurações de texto e botões
  );
}
