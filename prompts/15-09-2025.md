# Prompt: Autenticação e Multi-Empresa

**Data:** 15/09/2025
**Contexto:** Preocupação em como separar os dados de um restaurante de outro.

## 🗣️ Prompt do Usuário
"Como eu garanto que um restaurante não veja os dados do outro? Vou ter vários usuários usando o mesmo banco Supabase.
Crie o Provider de Autenticação que resolve isso."

## 🤖 Resposta (Lógica AuthProvider)

A solução é usar o `company_id`. Cada usuário logado pertence a uma empresa.

### Código: `lib/providers/auth_provider.dart`
*(Trecho da lógica implementada)*

```dart
class AuthProvider with ChangeNotifier {
  User? _user;
  String? _companyId;

  // Ao carregar o usuário, extraímos o ID da empresa dos metadados
  void _handleUserChange(User? user) {
    _user = user;
    if (_user != null && _user!.appMetadata.containsKey('company_id')) {
      _companyId = _user!.appMetadata['company_id'];
    }
    notifyListeners();
  }
  
  // O companyId será usado em TODOS os outros providers para filtrar dados.
}
