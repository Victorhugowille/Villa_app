# Prompt: Arquitetura do Bot de WhatsApp

**Data:** 23/10/2025
**Contexto:** Início do desenvolvimento de uma funcionalidade para automatizar avisos via WhatsApp usando Supabase.

## 🗣️ Prompt do Usuário
"Quero integrar um Bot de WhatsApp no meu restaurante para avisar quando o pedido saiu para entrega.
Estou usando Supabase. Devo fazer isso no código Flutter ou no Banco de Dados?
Crie a estrutura do `BotProvider` para lidar com isso."

## 🤖 Resposta (Estrutura Híbrida)

Recomendo usar **Supabase Edge Functions** para o envio real (segurança) e o Flutter apenas para o gatilho.

### Código: `lib/providers/bot_provider.dart`

```dart
class BotProvider with ChangeNotifier {
  final SupabaseClient _supabase = Supabase.instance.client;

  // Função chamada quando o motoboy sai com o pedido
  Future<void> notificarCliente(String pedidoId, String telefone) async {
    try {
      // Invoca a Edge Function que conecta com a API do WhatsApp (ex: Z-API)
      await _supabase.functions.invoke('whatsapp-notification', body: {
        'phone': telefone,
        'order_id': pedidoId,
        'status': 'out_for_delivery'
      });
    } catch (e) {
      debugPrint('Erro ao enviar notificação: $e');
    }
  }
}
