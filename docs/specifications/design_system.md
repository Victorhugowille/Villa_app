# Design System - VillaBistrô

## 🎨 Paleta de Cores

| Nome | Hex | Uso Principal |
| :--- | :--- | :--- |
| **Deep Green** | `#1E392A` | Fundo principal (Background), AppBars. Transmite sofisticação. |
| **Cream** | `#F0E6D1` | Textos, Ícones e Elementos sobre o fundo verde. Contraste suave. |
| **Gold / Bronze** | `#D4A373` | Botões de ação (CTA), Destaques, Bordas de seleção. |
| **Status Red** | `#B00020` | Indicador de Mesa Ocupada ou Erro. |
| **Status Amber** | `#FFC107` | Indicador de Pagamento Parcial (Atenção). |

## 🔤 Tipografia

**Família:** Roboto (Google Fonts)

* **H1 (Headline):** 24px, Bold. Usado para número da mesa e total da conta.
* **Body:** 16px, Regular. Usado para nomes de produtos.
* **Caption:** 12px, Light. Usado para observações do pedido.

## 📱 Componentes (Widgets)

* **Cards de Mesa:** Quadrados, com bordas arredondadas (Radius 12). Mudam de cor de fundo conforme o status.
* **Botões:** Estilo `ElevatedButton` com fundo *Gold* e texto *Deep Green*.
